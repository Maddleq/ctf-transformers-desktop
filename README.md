## О приложении

**CTF Transformers** — настольное приложение-табло для ведения игры «Захват флага»
(режим Conquest, как в Battlefield). Ведущий задаёт параметры игры, а по ходу матча
вручную отмечает захваченные точки — приложение само считает «жизни» команд и определяет
победителя.

Две команды: **Автоботы** (синие) и **Десептиконы** (фиолетовые). Кто контролирует больше
точек — тот медленнее теряет очки жизни. Когда у одной команды жизни кончаются — она
проигрывает.

### Как пользоваться

1. **Настройте игру** (верхний блок):
   - **Длительность игры** — общее время матча.
   - **Время начала** (+ дата, «мск») — время окончания посчитается автоматически.
   - **Всего точек в игре** — сколько точек на карте можно захватывать.
   - **Очки жизни команды** — стартовый запас жизней у каждой команды.
   - **Убывание за точку (очк/сек)** — сколько жизней теряется за каждую вражескую точку
     в секунду. Можно дробное число (например `0.5`) — чтобы игра шла дольше.
2. **Нажмите «▶ Старт»** — пойдёт таймер, кнопка заблокируется. Обе команды начинают
   терять жизни: чья команда контролирует меньше точек — теряет быстрее.
3. **По ходу игры** кнопками **+ / −** у каждой команды отмечайте захваченные точки.
   Когда все точки разобраны, нажатие **+** у одной команды отбирает точку у соперника.
4. **«⏸ Пауза»** — остановить таймер. На паузе можно поправить настройки и значения —
   они сохранятся, не сбросятся.
5. **Ручной ввод жизней.** Если учёт сбился, впишите текущее количество жизней прямо в
   поле «Очки жизни» у нужной команды.
6. **«Сброс»** — вернуть жизни и таймер к началу.
7. **Победитель** определяется, когда у одной из команд жизни дошли до нуля; внизу
   появится «Игра завершена» и итог.

### Логи игр

- Каждая завершённая игра записывается в **«Статистику прошлых игр»** (скрывающийся блок:
  дата, длительность, победитель). Внутри есть кнопка **«Очистить логи»**.
- **«Папка логов»** — выбрать папку на компьютере, куда сохранять отдельный JSON-файл на
  каждую игру (имя вида `log-2026-06-26-22-15-30-autobots.json` с полными данными матча).
- **«Экспорт логов»** — скачать всю статистику одним файлом-резервом.
- **«Импорт логов»** — загрузить ранее сохранённые файлы (можно несколько сразу) обратно
  в приложение.

> Все данные хранятся локально на вашем компьютере. Закрытие/обновление окна не теряет
> прогресс — игра продолжится с того же места.
